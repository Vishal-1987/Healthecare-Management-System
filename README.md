Healthcare SQL Data Analysis Project
📌 Description

This project focuses on analyzing a healthcare dataset using SQL. It covers basic to advanced SQL concepts including aggregations, joins, subqueries, views, stored procedures, window functions, and CTEs to derive meaningful insights.

🛠️ Tools & Technologies
SQL (MySQL)
Data Cleaning
Data Analysis
📂 Dataset Features

The dataset includes:

Patient details
Medical conditions
Admission & discharge dates
Billing amount
Doctor & hospital information
Insurance provider
🔍 Key Analysis Performed
Patient distribution by gender
Top 5 most common medical conditions
Total revenue generated by each hospital
Average hospital stay duration
Highest billing insurance provider
Most expensive medical condition
Conditions with above-average billing (Subquery)
Hospital-level billing insights using View
Admission-type analysis using Stored Procedure
Doctor ranking using Window Function
Above-average stay patients using CTE
⚙️ Advanced SQL Concepts Used
GROUP BY & Aggregations
Subqueries
Views
Stored Procedures
Window Functions (RANK)
Common Table Expressions (CTE)
Date Functions (DATEDIFF, STR_TO_DATE)
🚀 Sample Queries
-- Patient distribution by gender
SELECT Gender, COUNT(*) AS Total_Patient
FROM healthcare
GROUP BY Gender;

-- Top 5 medical conditions
SELECT `Medical Condition`, COUNT(*) AS commonly
FROM healthcare
GROUP BY `Medical Condition`
ORDER BY commonly DESC
LIMIT 5;

This project demonstrates the ability to perform end-to-end data analysis using SQL and extract actionable insights from healthcare data.
